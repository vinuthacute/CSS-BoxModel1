CSS BOX MODEL..!

Every element that can be displayed on a web page is comprised of one or more rectangular boxes. CSS box model typically describes how these rectangular boxes are laid out a web page. These boxes can have different properties and can interact with each other with differnt ways,but every box has a content area and optional surrounding padding,
